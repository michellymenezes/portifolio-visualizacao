Residentes paraibanas

published on February 15, 2018
Através de duas diferentes perspectivas... Para esta visualização foram utilizados dados dos setores censitários da PB segundo o censo do IBGE de 2010. Mais especificamente, os dados são um recorte relacionados às mulheres moradoras de domicílios particulares e coletivos. A visualização abaixo está representado o número de mulheres em cada setor censitário do estado. A variação das cores representa um intervalo em valores brutos. É possível observar que a maior parte se encontra com números no intervalo 0-249 ou 250-499. Read More...
